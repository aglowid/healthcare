# README

* Ruby version
  ruby - 2.4.0

* System dependencies

* Configuration
  - clone project first with following command:
    git clone

    run - bundle install

* Database creation
  Database - postgres
  1) To create database
    rake db:create
  2) To migrate Database
    rake db:migrate
  3) To add default entries
    rake db:seed

	And start server using :-
  => rails s
